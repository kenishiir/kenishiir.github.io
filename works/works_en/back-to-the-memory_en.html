Back to the Memory

A product that can trace and play memories.

The name of this product is "anywhere stand". With "anywhere stand" you can fix the smartphone anywhere in the air. By using this like a document camera, it can record memos and sketches. By doing this, we can rewind and reproduce what kind of thought processes existed up to the conclusion as to how ideas were born.
Also, designers and architects who communicate using drawings and sketches can proceed more smoothly by using 'anywhere stand' when making video meetings.
