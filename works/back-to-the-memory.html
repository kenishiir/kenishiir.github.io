Back to the Memory

記憶を遡り、思考を再生するプロダクト。

このプロダクトの名前は「どこでもスタンド」。「どこでもスタンド」を使えばスマホをどこでも空中に固定することができる。これを書画カメラのように使用することで、自分のメモや覚書きを録画することができる。これにより、どのようにしてアイディアが生まれたか、結論に至るまでどのような思考プロセスがあったかを巻き戻し、再生することができる。
また、図やスケッチを用いて議論を進めるデザイナーや建築家は、ビデオ通話をする際に「どこでもスタンド」を使うことで、よりスムーズにミーティングを進めることができる。
